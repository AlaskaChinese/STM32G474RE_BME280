# Nucleo-G474RE with BME280
 BME280 project for nucleo-g474re with i2c and lpuart.

## Wire connection
| Ports | STM32 |
| :---: | :---: |
| BME280_SDA | PC9 |
| BME280_SCL | PC8 |
| OLED_SDA | PA8 |
| OLED_SCL | PA9 |
